# Journal Match: Road to Acceptance 🎓🔥

A satirical, swipe-based browser game (à la Tinder) about the agonizing "Publish or Perish" reality of academia. Players take on the role of a PhD student trying to publish 3 papers before their funding runs out (6 in-game years), navigating exorbitant APCs, predatory publishers, and Reviewer #2.

## 🚀 Features
*   **Single-file architecture:** Entirely contained in one `index.html` file (HTML, CSS, JS). Perfect for GitHub Pages.
*   **Real-time Swiping:** Procedurally generated journals with dynamic stats (APCs, Impact Factors, Review Times).
*   **Dynamic Certificate Generation:** Creates a downloadable, personalized pixel-art PhD certificate upon winning using HTML5 Canvas.
